# EffectiveObjective-C-2.0

Objective-C 指针类型的变量有两个：一个是编译时类型，一个是运行时类型；如果编译时类型和运行时类型不一致的时候就可能出现多态。
相同类型的变量调用同一个方法呈现出多种不同的行为特征就是多态
多态可以提高程序的可拓展性
